Zásady ochrany údajů

Tyto zásady popisují, jak server SirTros-city pracuje s technickými informacemi hráčů při připojení na Minecraft server. Nejde o webovou službu ani o marketingové sledování.

Shromažďované údaje

Při připojení na server může být použito Minecraft jméno hráče a IP adresa pouze pro technické zajištění připojení.

Účel použití

Údaje slouží pro fungování serveru, stabilitu připojení, ochranu proti poškozování světa a správu hráčů.

Ukládání údajů

Veškeré technické informace jsou dočasné a automaticky se vymazávají po vypnutí serveru. Neprobíhá dlouhodobé ukládání dat.

Důležité informace

SirTros-city je komunitní Minecraft server a nepoužívá žádné externí sledování ani reklamy.
